Visualization of 750000 butterflies from the Natural History Museum

This is an interactive data visualization project that shows the collection of lepidoptera of the Natural History Museum in London. It uses machine learning methods to process the data, including deep neural networks and t-SNE to arrange the specimens by visual similarity. Click here to read a detailed explanation of the data preparation process.
The source code for this project is available on Github. This project was created by Marian Kleineberg. Follow me on Twitter.
